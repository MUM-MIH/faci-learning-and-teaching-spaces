[fa-share-alt-square fa-lg] Hybrid Sharing Options

Auditorium 1 is able to stream the Lectern PC onto the projector screens while simultaneously streaming another device. Here is how to configure this option.

Note: This is an Advanced Module. Do not hesitate to rewatch the video to understand the functionality of this Hybrid Sharing Option.

MIMO (Multiple Input Multiple Output)

The Hybrid sharing option, MIMO (Multiple Input Multiple Output) can be set up in the following configurations.
  • BYOD (Bring Your Own Device)  - lectern combination

  1. Connect your own device following the BYOD instructions.
  2. Select Presentation on the toolbar.
  3. For video: Select PC, then select the lectern computer as the input. Select Projector 1 as the output. The video feed from the lectern computer is now routed to Projector 1.
  4. For audio: Select either Laptop or PC as your audio source. Select Audio as the output. The audio feed from the selected device is now routed to the Auditorium speakers.

The hybrid sharing option can be set up in a lectern - document camera combination, BYOD -  document camera combination.
